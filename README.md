# 🌌 Celestial Catch – Hand Tracking XR Game

An interactive browser-based XR experience that uses **AI hand tracking** to control gameplay. Built using **Three.js** and **MediaPipe Hands**, this project demonstrates real-time gesture interaction in a 3D environment.

---

## 🎮 Features

- Real-time hand tracking control  
- Catch falling celestial objects (stars)  
- Dynamic difficulty (speed increases with score)  
- Lives system & scoring HUD  
- 3D visuals with lighting and fog effects  
- Smooth gameplay with physics-based interactions  

---

## 🛠️ Technologies Used

- Three.js – 3D rendering  
- MediaPipe Hands – Hand tracking (AI)  
- JavaScript (Vanilla)  
- HTML5 & CSS3

## 🎯 Controls

- Move your hand left/right → controls the bucket  
- Catch falling objects to score points  
- Missing objects reduces lives
